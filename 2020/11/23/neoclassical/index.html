Resolvendo o Modelo de Crescimento Neoclássico

2020-11-23

A dinâmica do modelo de crescimento neoclássico

Neste post, pretendo encontrar a solução para a dinâmica do modelo de crescimento neoclássico, isto é, a regra para as sequências \(\{k_t\}_{t = 0}^\infty\) e \(\{c_t\}_{t = 0}^\infty\) dado um valor inicial \(k_0 > 0\).

O problema

O modelo de crescimento neoclássico tem solução analítica quando \(\delta = 1\), isto é, quando todo o capital se deprecia de um período para o outro, então vamos nos limitar a este caso. Considerando uma função de utilidade genérica \(u(c)\) e uma função de produção genérica \(f(k)\) que obedecem as condições de Inada, o problema pode ser escrito como:

\[ \begin{aligned} \max_{\{c_t\}^{\infty}_{t = 0}} \quad & \sum_{t=0}^{\infty}{ \beta^{t} u(c_t) }\\ \textrm{s.t.} \quad & k_{t+1} \leq g(c_t, k_t) \equiv f(k_t) - c_t, \quad \forall t\\ & c_t \geq 0, \quad k_t \geq 0, \quad \forall t \\ & k_0 > 0 \quad dado \\ \end{aligned} \]

O princípio do máximo de Pontryagin

Para resolver este problema, vamos usar o princípio do máximo de Pontryagin, que pede a definição uma função auxiliar conhecida como Hamiltoniano (no tempo \(t\)):

\[ H_t(k_t, c_t, \lambda_{t+1}) \equiv \beta^t u(c_t) + \lambda_{t+1}g(k_t, c_t) \]

As condições para um ótimo são:

Equação adjunta: \[\lambda_t^* = \nabla_k H_t(k_t^*, c_t^*, \lambda_{t+1}^*)\] Princípio do máximo fraco: \[ \nabla_c^* H_t(k_t^*, c_t^*, \lambda_{t+1}^*) = 0 \] Folga complementar: \[ \lambda_{t+1}^*\big[ g(k_t^*, c_t^*) - k_{t+1}^* \big] = 0 \]

Encontrando a solução

De cada uma das condições, temos, respectivamente:

\[\begin{align} \lambda_t = \lambda_{t+1} f'(k_t) + \beta^t 0 \tag{1} \end{align}\] \[\begin{align} -\lambda_{t+1} + \beta^t u'(c_t) = 0 \tag{2} \end{align}\] \[\begin{align} \lambda_{t+1}(f(k_t) - c_t - k_{t+1}) = 0 \tag{3} \end{align}\]

Resolvemos as equações para o caso \(u(c_t) = ln c_t\) e \(f(k_t) = k_t^\alpha\).

De (2), temos: \[ \lambda_{t+1} = \beta u'(c_t) \]

Substituindo em (3): \[\begin{equation} \beta u'(c_t) = \beta u'(c_{t+1}) f'(k_{t+1}), \quad \forall t \tag{4} \end{equation}\]

Como \(\lambda_t > 0 \quad \forall t\), temos também:

\[\begin{equation} k_t^\alpha - c_t - k_{t+1} = 0, \quad \forall t \tag{5} \end{equation}\]

Substituindo (5) em (4):

\[ k_t^\alpha - k_{t+1} = \frac{k_{t+1}^\alpha + k_{t+2}}{\beta \alpha k_{t+1}^{\alpha-1}} \]

Isolando \(k_{t+2}\): \[\begin{align*} k_{t+2} & = k_{t+1}^\alpha - \beta \alpha (k_t^\alpha - k_{t+1})k_{t+1}^\alpha \\ & = k_{t+1}^{\alpha-1}[k_{t+1} - \beta \alpha k_t^\alpha] + \beta \alpha k_{t+1}^\alpha \end{align*}\]

Observe que, se \(k_{t+1} = \beta \alpha k_t^\alpha\), o termo entre colchetes é igual a zero e ficamos apenas \(k_{t+2} = \beta \alpha k_{t+1}^\alpha\), que é a mesma equação, mas para o tempo \(t+1\).

O modelo, portanto, reduz para a equação em diferenças de primeiro grau: \[ k_{t+1} = \beta \alpha k_t^\alpha \]

De cuja solução é simplesmente: \[ k_t = \exp \bigg[ \frac{(\alpha^t - 1)(ln\beta + ln\alpha)}{\alpha - 1} \bigg] + A \alpha^{t - 1} \]

Onde \(A\) é uma constante que pode ser determianda a partir do valor inicial \(k_0\):

\[\begin{align*} k_0 &= \exp \bigg[ \frac{(\alpha^0 - 1)(ln\beta + ln\alpha)}{\alpha - 1}\bigg] + A \alpha^{0 - 1} \\ &= 1 + \frac{A}{\alpha} \\ \\ A &= \alpha(k_0 -1) \end{align*}\]

Podemos encontrar a regra para a sequência \(\{c_t\}_{t = 0}^\infty\) a partir da equação (5).

A solução do modelo de crescimento neoclássico é, portanto:

\[ \left\{ \begin{array}{>{\displaystyle\tallstrut}l@{}} k_t = \exp \bigg[ \frac{(\alpha^t - 1)(ln\beta + ln\alpha)}{\alpha - 1} \bigg] + (k_0 -1) \alpha^{t} \\ c_t = \bigg( \exp \bigg[ \frac{(\alpha^t - 1)(ln\beta + ln\alpha)}{\alpha - 1} \bigg] + (k_0 -1) \alpha^{t} \bigg)^\alpha - \exp \bigg[ \frac{(\alpha^{t+1} - 1)(ln\beta + ln\alpha)}{\alpha - 1} \bigg] + (k_0 -1) \alpha^{t+1} \end{array} \right. \]
